Malloc

Introduction to Computer Systems (15-213)

Fall 2015

A dynamic memory allocator based on the malloc/free library in C. Was optimized for both space and efficiency using an underlying segmented free list with best-fit search. Utilized many important C programming concepts such as pointer arithmetic/casting, bit arithmetic, and preprocessor macros.
